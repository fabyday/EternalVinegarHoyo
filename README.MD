# 밥똥이리오효 



맥에서 밥먹고 이리하고 똥싸고 호요게임 하는데 특화된 런처임

__쓰다가 영정먹어도 나는 죄가 없어요(매우 중요)__




## 설치

Release 들어가서 설치  




## 신선한 와인과 안주 제공 단체

[DXMT](https://github.com/3Shain/dxmt)
:Metal Layer를 바로 이용하는 DX12(강함)

[Official wine](https://www.winehq.org/)
:"그냥" 와인

[dawn winary](https://dawn.wine/dawn-winery)
: 아니메 스타일 게임용(?) 와인

[Jedite](https://codeberg.org/mkrsym1/jadeite)
: 안티치트 패치




## 카탈로그 관리

__Wine__ 카탈로그

- **Github Repo 카탈로그(주인장이 관리하는)** : src/Common/Constant/WineCatalog.ts 
  - 하드 코딩 되어 있음 변경 불가
- **Gist or custom Github Repo** : 누군가가 관리하거나 주인장이 관리하는 래포
  - 외부에 관리하는거라 파일만으로 버전업데이트 됨 
- **user defined** : 사용자나 아무나 관리하는 카탈로그 
  - 모름 알아서 관리 잘 되겠지
